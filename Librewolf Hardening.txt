LIBREWOLF HARDENING GUIDE

about:config



TRACKING & TELEMETRY

beacon.enabled
Change: true -> false
Description: Disables web beacons. Stops sites from sending hidden
background pings for tracking.

network.trr.confirmation_telemetry_enabled
Change: true -> false
Description: Stops telemetry about DNS-over-HTTPS. No reporting to
Mozilla servers.

nimbus.telemetry.targetingContextEnabled
Change: true -> false
Description: Disables feature experimentation telemetry. Avoids targeted
configuration experiments.

telemetry.fog.init_on_shutdown
Change: true -> false
Description: Prevents telemetry framework initialization during shutdown.
Less data logging.

datareporting.policy.dataSubmissionEnabled
Change: true -> false
Description: Stops all background data submission. No automatic telemetry
uploads.

toolkit.telemetry.cachedProfileGroupID
Action: DELETE
Description: Deletes telemetry ID. Prevents persistent identification of
your profile.

toolkit.telemetry.dap.helper.hpke
Action: DELETE
Description: Removes telemetry encryption helper. Disables data submission
support.

toolkit.telemetry.dap.helper.url
Action: DELETE
Description: Deletes telemetry endpoint. Ensures no telemetry destination.

toolkit.telemetry.dap.leader.hpke
Action: DELETE
Description: Removes remaining telemetry encryption parameters. Blocks
telemetry communication entirely.



SENSORS & DEVICE INFORMATION

device.sensors.enabled
Change: true -> false
Description: Blocks access to all motion/orientation sensors. Prevents
motion-based fingerprinting.

device.sensors.motion.enabled
Change: true -> false
Description: Specifically blocks motion sensors. Stops tracking of device
movement.

device.sensors.orientation.enabled
Change: true -> false
Description: Disables orientation sensors. Stops detection of screen tilts
or angles.

dom.battery.enabled
Change: true -> false
Description: Hides battery information. Prevents sites from fingerprinting
device power data.

dom.gamepad.enabled
Change: true -> false
Description: Blocks Gamepad API. Stops connected controllers from leaking
unique hardware data.



WEBRTC & NETWORK

media.peerconnection.enabled
Change: true -> false
Description: Disables WebRTC. Stops IP leaks through video/audio calls.
NOTE: Turn on if you use Discord or Meet.

network.captive-portal-service.enabled
Change: true -> false
Description: Stops the constant "pings" to Mozilla servers every time you
connect to a network.

browser.places.speculativeConnect.enabled
Change: true -> false
Description: Stops background connections that predict your keystrokes.



GEOLOCATION

geo.enabled
Change: true -> false
Description: Disables geolocation. Prevents websites from detecting your
physical location.

geo.prompt.open_system_prefs
Change: true -> false
Description: Stops Firefox from opening OS location settings. Fully
disables location prompts.

geo.provider.network.url
Action: DELETE
Description: Removes Google geolocation service URL. Ensures no background
geolocation requests.



CLIPBOARD & INPUT PRIVACY

dom.event.clipboardevents.enabled
Change: true -> false
Description: Prevents websites from detecting copy/paste/cut actions.
Enhances privacy on input fields.
HTTPS & SECURITY

dom.security.https_only_mode_ever_enabled
Change: false -> true
Description: Forces all connections through HTTPS. Blocks insecure (HTTP)
connections.



TRACKING PROTECTION

privacy.antitracking.isolateContentScriptResources
Change: false -> true
Description: Isolates scripts per site. Prevents trackers from sharing
cached resources across domains.

privacy.trackingprotection.allow_list.convenience.enabled
Change: true -> false
Description: Ensures tracking protection applies everywhere. No exceptions
for "trusted" sites.

privacy.trackingprotection.enabled
Change: false -> true
Description: Enables global tracking protection. Blocks known tracking
domains.

privacy.trackingprotection.socialtracking.enabled
Change: false -> true
Description: Blocks social media trackers. Protects against Facebook,
Twitter, etc. embedding tracking scripts.



REFERER HEADERS

network.http.referer.XOriginPolicy
Setting: 1
Description: Only sends "Referer" headers when the hostnames match. Stops
cross-site tracking.

network.http.referer.XOriginTrimmingPolicy
Setting: 2
Description: Strips the "Referer" header to just the domain. Removes
specific page paths.



NORMANDY (REMOTE SETTINGS)

app.normandy.enabled
Change: true -> false
Description: Stops remotely changing your browser settings or installing
experimental add-ons without your explicit permission.

app.normandy.api_url
Action: DELETE
Description: This is the web address the browser checks to receive the
instructions mentioned above.
